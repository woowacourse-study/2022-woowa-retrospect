## 💛 회고 방식
1. 일요일 오전 10시에 모입니다.
2. 20분 동안 한 주를 돌아보며, 좋았던 점 아쉬웠던 점을 감정회고/기술회고 측면으로 작성합니다.
3. 툴은 Figjam을 이용합니다.
4. 일요일이 지나기 전에, 개인회고 링크를 README에 올립니다.
5. 한 주에 위 규칙을 지키지 못 할 시 5000원의 벌금을 냅니다.
